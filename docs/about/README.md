#  关于项目

## 初衷

HarukaBot 是我出于学习目的，利用课余时间制作出来的。希望能为大家提供一个优雅的推送方式。

但是介于本人技术力与精力有限，HarukaBot 仍然存在诸多不足，远远没有达到我对它的期望。

## 历程

从 2020 年 3 月份至今，超过大半年的不断更新，目前 `HarukaBot v1` 也仅仅只是达到了稳定可用。距离我心目中的好用、易用，仍有很长的路要走。

## 未来

按照我原本的计划 `v1` 阶段是完善 HarukaBot 的基础功能。确保它能尽可能的稳定运行，遇到错误也有完善的应对机制。目前这部分已经实现的差不多了。

`v2` 阶段是制作 B站 的登录方法，可以让 HarukaBot 绑定 B站 账号，提高爬虫的频率与稳定性。

`v3` 阶段是对所有功能的细节进行优化，比如更友好的交互指令，更人性化的推送设计。当然也包括搭建文档网站，简化部署流程，至少我原本是这么计划的。

但是如你所见，原本 `v3` 中的部分内容我甚至在 `v1` 阶段就已经提前做出来了。因此我估计不会真正的有一个 `v3`， 这些对细节的打磨应该会穿插在我接下来的更新中。~~即使真的有恐怕也是我又开了个别的坑。~~

目前为了能更高效的开发 `v2`，正在对 `v1` 中的部分代码进行重构。

## 坚持

我也不知道这个凭着我一腔热情的项目可以持续多久。

维护这个项目，可能是我目前为止坚持时间最久的一件事了。我从来就不是一个擅长坚持的人，能走到这一步我自己说实话都有点意外。

我也希望能继续做下去，但是谁知道我的热情会不会在达到我对它的期望之前燃尽呢（苦笑）。
